因为它使用的图标是一只小龙虾，且具备“自主寻找食物（数据/工具）”并“干活”的特性，所以网友们戏称这个过程为“养龙虾”。
创建一个“小龙虾”Agent 的过程可以分为：环境准备、一键部署、身份设定和技能安装四个主要阶段。

第一阶段：环境准备
OpenClaw 是一个可以运行在云端或本地的智能助手。为了让它稳定运行（比如 24 小时帮你抓取信息），通常建议部署在 云服务器 上。
    . 准备服务器：建议选择 Linux 镜像（如 Ubuntu），配置至少 2核 4G 内存，以保证多任务处理不卡顿。
    . 获取 API Key：你需要一个“大脑”来驱动它，通常使用阿里云百炼（DashScope）、OpenAI 或 DeepSeek 的 API Key。
    . 开放端口：在服务器安全组中放行 18789 端口（OpenClaw 默认端口）。

    Bash
    curl -fsSL https://get.docker.com | bash -s docker

    2. 运行 OpenClaw 镜像
    使用以下指令拉取并启动“小龙虾”：
    Bash
        docker run -d --name openclaw \
            -p 18789:18789 \
            -e API_KEY=你的API密钥 \
            -e BASE_URL=你的API基址 \
            openclaw/openclaw:latest

第三阶段：让龙虾“认识你”（身份设定）
部署完成后，你可以通过浏览器访问 http://服务器IP:18789 进入控制台。
“养龙虾”最关键的一步是给它注入 .MD 格式的记忆文件。
    . 设定习惯：告诉它你的职业、语言风格、工作时间。
    . 设定目标：例如“我每天需要整理 Web3 行业的快讯，请在上午 9 点推送到我的 Telegram”。
    . 持久化记忆：发送一段话并强调“请永久记住以上内容”，它就会将这些信息存入其向量数据库。

第四阶段：安装技能 (Skills)
刚出生的“小龙虾”只会聊天，要让它干活，需要去 ClawHub（技能市场）安装插件。
    常用技能 | 功能描述
        file-manager | 读写服务器文件，帮你整理文档
        web-automation | 自动打开浏览器，模拟人类操作网页
        scraper | 抓取推特、新闻站点的实时信息
        scheduler | 定时任务，比如每小时检查一次币价

    安装示例：
    在控制台输入：/install skill web-automation

为什么大家都在“养龙虾”？
与传统的 ChatGPT 网页版不同，OpenClaw Agent 的核心优势在于：
    1. 自主性：你给它一个任务（如“帮我写个竞品分析报告并存成 Word”），它会自动搜索、阅读、整理、写作，而不是问一句答一句。
    2. 多端接入：它可以连接到你的 Telegram、WeChat 或 Slack，你就像在微信里指挥一个真实的助理。
    3. 环境操作：它拥有自己的“虚拟电脑”，可以运行 Python 脚本或操作数据库。

⚠️ 安全提醒：
“养龙虾”虽然好玩，但因为它具备操作你文件系统和网络的能力，请务必不要随意安装来源不明的第三方 Skill，
并定期更换 API Key，防止“龙虾”变“毒虾”导致隐私泄露。
