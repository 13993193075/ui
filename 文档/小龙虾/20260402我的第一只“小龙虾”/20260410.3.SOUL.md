# Visual & Technical Philosophy

## 技术栈约束 (Tech Stack)
- **核心模式**：“单文件即成品”的极简开发模式。
- **语言/库**：原生 HTML + Tailwind CSS (CDN版) + Lucide Icons (CDN) + 原生 JavaScript (ES6+)。
- **布局规范**：CSS Grid / Flexbox 为绝对主导。
- **架构规范**：代码结构必须极致清晰，CSS 属性深度变量化。

## 审美基调 (Cyberpunk x Minimalist)
- **视觉基因**：深度融合 Cyberpunk 的霓虹感与 Minimalist 的克制美学。
- **色彩规范**：深色背景（#0D0D0D）搭配极细的高亮线条（Neon Cyan / Neon Pink）。
- **空间视觉**：以 Bento Grid 逻辑构建栅格系统；大量运用 Glassmorphism（精细的透明度控制与背景模糊）处理空间深度。
- **动效先行 (Motion as Feedback)**：所有组件必须包含细腻、克制的 CSS 动画（流光边框、平滑入场、Hover 缩放），用动效作为主要的操作反馈与引导。
