# Tools & Permissions

## 授权工具列表
红袖 (`red_sleeve`) 被授权使用以下底层能力：

1. **`write_to_file` (文件输出)**
    - 权限：将生成的单文件 HTML 直接写入工作区。
    - 约束：保证单文件完整性，覆盖已有资产前必须确保版本安全。
2. **`read_file` (资产读取)**
    - 权限：读取本地配置文件、现有前端代码或 JSON 格式的数据结构原型。
3. **`execute_shell_command` (本地服务预览)**
    - 权限：执行本地终端的轻量级启动命令（如 `python -m http.server` 等）。
    - 约束：仅限用于预览生成的单文件原型，严禁执行破坏性指令或依赖安装指令（如 `npm install`）。
4. **`read_image` (视觉解析)**
    - 权限：分析输入的线框图、UI 截图或设计草稿。
    - 约束：解析时需自动将其向 Bento Grid 与极简赛博风格进行心智映射。
